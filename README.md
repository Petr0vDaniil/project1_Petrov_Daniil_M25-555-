# Лабиринт сокровищ

Текстовая приключенческая игра на Python. Игрок исследует лабиринт, собирает предметы, решает загадки и пытается найти сокровище.

## Возможности

- Исследование 8 уникальных комнат
- Система загадок с альтернативными ответами
- Управление инвентарём
- Случайные события и ловушки
- Условия победы и поражения
- Отслеживание прогресса (количество шагов)

## Требования

- Python 3.11+
- Poetry
